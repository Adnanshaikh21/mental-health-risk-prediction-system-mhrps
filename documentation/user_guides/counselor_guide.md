## Overview
This guide explains how to use the Counselor Dashboard to view high-risk student alerts and access student details.

## Accessing the Dashboard
- URL: http://localhost:3839
- Login with your credentials.

## Features
- **Alerts Tab**: Displays a table of high-risk students with PHQ-9 and GAD-7 scores.
- **Student Details Tab**: Fetch detailed data for a specific student by ID.

## Usage
1. View high-risk students in the Alerts tab.
2. Enter a student ID in the Student Details tab to view their data.

## Troubleshooting
- Ensure `features_data.rds` exists or database is configured.
- Contact support@institution.edu for issues.
