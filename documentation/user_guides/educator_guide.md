## Overview
This guide explains how to use the Educator Dashboard to monitor student mental health risks and submit data for predictions.

## Accessing the Dashboard
- URL: http://localhost:3838
- Login with your credentials.

## Features
- **Overview Tab**: Displays counts of students by risk level (High, Medium, Low) and a risk distribution plot.
- **Student Input Tab**: Enter student data to predict mental health risk.

## Usage
1. Navigate to the Student Input tab.
2. Enter data (e.g., semester, GPA, PHQ-9, GAD-7).
3. Click "Predict" to view the risk level and probabilities.

## Troubleshooting
- Ensure `model.rds` exists in `model/output`.
- Contact support@institution.edu for issues.
