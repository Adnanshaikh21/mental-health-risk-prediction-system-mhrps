## Overview
This guide explains how to use the Admin Dashboard to monitor system performance and model metrics.

## Accessing the Dashboard
- URL: http://localhost:3840
- Login with admin credentials.

## Features
- **System Overview Tab**: Displays model accuracy, API status, and data quality metrics.
- **Performance Metrics Tab**: Shows the confusion matrix and its plot.

## Usage
1. Check system health in the System Overview tab.
2. Review model performance in the Performance Metrics tab.

## Troubleshooting
- Ensure `model.rds` and `test_data.rds` exist.
- Contact support@institution.edu for issues.
